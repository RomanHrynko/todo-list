### Todo-List

Todo-List will allow you to plan your day and have time for everything

### You can see the link [Todo-list](https://priceless-raman-141258.netlify.app)

- [BEM methodology](https://en.bem.info/methodology/)
- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/)
